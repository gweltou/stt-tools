1. Aller dans le dossier ~/STT/stt-tools et ouvrir un terminal avec la touche "F4"

2. Ouvrir l'éditeur wavsplit en tapant dans le terminal "./wavsplit" suivi du nom du fichier audio (au format .wav) sur lequel travailler.
   Note : Si le fichier audio est dans le dossier parent du dossier actuel il faudra taper "./wavesplit ../noz_an_nedelegetcetcetc.wav"
          Les caractères "../" avant le nom du fichier indiquent au terminal de rechercher le fichier dans le dossier parent du dossier actuel.
          Plutôt que de taper le nom du fichier entièrement, on peut écrire le début du nom du fichier et appuyer sur la touche TAB pour compléter le nom du fichier automatiquement.

3. Ouvrir également le fichier texte du même nom que le fichier audio.
   Activer la correction automatique en breton de l'éditeur de texte en cliquant sur l'icone "menu" (à droite du bouton "enregistrer"), puis "outils" > "Mettre en évidence les mots mal orthographiés" (case à cocher). Cliquez ensuite sur "définir la langue" dans le même menu et choisir "breton".
   Toutes les modifications du texte se font dans l'éditeur de texte, et doivent être enregistrées (touches CTRL + s) pour qu'elles apparaissent dans l'outil "wavesplit".

Utiliser le raccourci clavier ALT + TAB pour passer d'une application à l'autre.

4. Utilisation de l'éditeur "Wavsplit" :

 <ENTER> pour lire le segment actuel
 n <ENTER> pour lire le segment suivant
 j <ENTER> pour joindre le segment actuel avec le segment précédent
 z <ENTER> pour annuler la dernière modification
 s <ENTER> pour sauvegarder les modifications
 h <ENTER> pour afficher l'aide
